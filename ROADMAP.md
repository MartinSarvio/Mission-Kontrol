# Mission Kontrol — Roadmap

## Fase 1: Agent Organisation & Synlighed (PRIORITET)
1. **Agenter-siden redesign**
   - Organisationsdiagram (Maison → team)
   - Online/inaktiv/arbejder status per agent
   - Budget-ramme per agent (token forbrug vs. grænse)
   - Ansvarsområder tydeligt vist
   
2. **Kommunikationstab**
   - Se agents tale sammen i real-time
   - Diskussioner linker automatisk til Opgaver
   - Hub-view med alle agent-samtaler

3. **Opgaver-siden forbedring**
   - Fuld session-historik med samtaler og tool calls
   - Momentum-baseret ranking af tasks
   - Status: kø → aktiv → review → færdig
   - Hvem arbejder på hvad lige nu

## Fase 2: Intelligens & Automation
4. **Intelligens-siden (suggestions)**
   - Web/Twitter scanning for use cases
   - Email-lignende view med forslag
   - "Deploy" knap → sender til kø/workshop
   - Snapshot af hvad det ville betyde for FLOW

5. **Dokument Digest**
   - Upload PDFs
   - Hurtig processing og opsummering
   - Viden integreres i agent-hukommelse

6. **Architect Agent**
   - Automatisk audit af Mission Kontrol hver 24 timer
   - Find bugs, foreslå forbedringer
   - Arbejder mens Martin sover

## Fase 3: Telegram Integration
7. **Agent-switching guide**
   - Hvordan tale med specifikke agents via Telegram
   - Bindings setup per agent
   - Guide synlig i appen

## Prioritering
- Fase 1 FØRST — synlighed og organisation
- Fase 2 når Fase 1 virker
- Fase 3 parallel med Fase 2
